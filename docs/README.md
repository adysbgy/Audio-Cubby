# Dokumentasi

Folder ini berisi semua catatan utama untuk paket audio Cubby.

| File | Isi |
|---|---|
| `CUBBY_AUDIO_HANDOFF.md` | Konteks utama untuk chat baru |
| `CHAT_PROMPT.md` | Prompt singkat untuk chat baru |
| `audio_index.md` | Daftar file audio dan status tiap asset |
| `candidate_audit_report.md` | Ringkasan kandidat, blocker, dan status reset |
| `review_notes.md` | Catatan kecocokan audio dengan arah game |
| `source_links.md` | Tabel source, creator, lisensi, dan download URL |
| `license_notes.md` | Ringkasan sumber dan lisensi |
| `final_audit.md` | Audit akhir dan status reset |
| `manual_listening_audit.md` | Checklist audit dengar manual |
| `technical_audio_audit.md` | Audit teknis durasi, peak, dan file integrity |
